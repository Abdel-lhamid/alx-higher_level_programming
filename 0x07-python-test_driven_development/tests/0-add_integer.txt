# run test with python3 -m doctest -v ./tests/0-add_integer.txt

let's Import the methode to test first
>>> add_integer = __import__('0-add_integer').add_integer

Success Cases:

simple example that will pass the tests with a,b int:
	>>> print(add_integer(2, 3))
	5
	>>> print(add_integer(-1, 5))
	4
a and/or b are float:
	>>> print(add_integer(-3.2, 7.4))
	4
	>>> print(add_integer(15.7, 1000))
	1015
testing default value of b=98:
	>>> print(add_integer(12))
	110

Failed Cases:

	Testing void args:
	>>> print(add_integer(None,0))
	Traceback (most recent call last):
	...
	TypeError: a must be an integer

Test more args:
	>>> print(add_integer(1, 1, 1)):
	Traceback (most recent call last):
	...
	TypeError: add_integer() takes from 1 to 2 positional arguments but 3 were given
